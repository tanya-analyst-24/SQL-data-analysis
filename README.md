# SQL-data-analysis
Customer Data Analytics for Identity Digital utilizing SQL
Developed complex SQL queries using joins, aggregations, and subqueries to analyze and manage customer domain data.
Utilized Google BigQuery for efficient processing and querying of large-scale datasets.
Conducted customer trend analysis to identify high-value customers and domain usage patterns.
Optimized SQL queries to improve performance and reduce execution time.
Generated insights from data to support data-driven decision-making across internal teams.
Worked with structured datasets to analyze domain registrations, usage trends, and customer behavior.
